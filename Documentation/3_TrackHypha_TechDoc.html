Documentation technique¶

Module Mushroom¶

Synopsis:

This module represents and manages the structure of the mycelium extracted from a data set throught these 5 classes:

  • Coordinates
  • HyphaSegment
  • Analysis
  • Project
class Mushroom.Analysis(startApex, startImg)[source]¶

Class representing an analysis run on an image dataset in fonction of the user’s parameters.

Paramètres:
  • id (int) – the id of the analysis
  • startApex (Coordinates) – the Coordinates of the apex chosen as the starting point of the analysis
  • startImg (int) – the index of the start image of the analysis
  • endImg (int) – the index of the end image of the analysis
  • steps (dict{int : [Coordinates, int]}) – a dictionary of the analysis steps, linking the step index with the Coordinates of the followed apex and the index of the corresponding image
  • finalImg (Image) – an image showing the hyphae that have been explored by the analysis
  • stepImg (Image) – an image showing the position of the explored apex for the current step
  • currentStep (int) – the step currently being displayed
  • previousStepDisplayed (int) – the step previously being displayed
  • segments (list[HyphaSegment]) – a list of all the HyphaSegment created by the analysis
  • hyphae (dict{int : list[int]}) – a hyphae dictionary linkink each hypha with the id of its segments
  • list_pixels (list[Coordinates]) – a list of Coordinates of the hyphae pixels
  • processingTime (int) – the processing time of the analysis

class Mushroom.HyphaSegment(previousID, start)[source]¶

Class representing a hypha segment which is the section of the hypha comprised between two nodes or between a node and an apex.

Paramètres:
  • id (int) – the id of the segment
  • previous (int) – the id of the previous segment
  • deadEnd (bool) – True if the segment is the last one of the hypha, False otherwise
  • coord (list[Coordinates]) – the start and end Coordinates of the segment
  • evolution (dict{int : [Coordinates, int]}) – a dictionary of the evolution steps of the segment, linking the relevant image with the segment apex Coordinates and its length
  • size (int) – size of the hypha segment in pixel

Module AI¶

Synopsis:The module AI enables the analysis on a given set of data. It allows the following of the evolution of an apex and all its hyphae daughters, the cutting into segments, the size calculation of each segment and the composition of all the hyphae. It also saves the id number of the image and the coordinates of the followed apex during each step in order to make it easier to visualize it on the user interface.

Renvoie:

a list of all the white pixels coordinates of the area

list[Coordinates]
